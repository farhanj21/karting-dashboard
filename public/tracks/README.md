# Track Logos

Place track logo images in this folder.

## Naming Convention

Logo files should be named using the track slug:
- `sportzilla-formula-karting.png`
- `apex-autodrome.png`

Supported formats: PNG, JPG, JPEG, SVG, WEBP

## Recommended Specifications

- Format: PNG with transparent background (preferred) or SVG
- Size: 200x200px minimum
- Aspect ratio: Square or maintain original aspect ratio
